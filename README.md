# CalendarNoteApp
The application allows to select date and save notes in Jetpack Compose.

## Technologies and Libraries Used
- **Kotlin**
- **Jetpack Compose**
- **Room Database**
- **Dependency injection with Hilt**
- **MVVM**
- **Clean Architecture**
- **Navigation**
- **Coroutines**
